```

```

## 왜 이런 일이 일어났나?

### 🔍 원인 분석

```
김할머니의 성격:
├─ 감정 표현 욕구: 80점 (따뜻한 대화 원함)
├─ 활동 선호: 30점 (조용히 쉬고 싶음)
├─ 인내심: 85점 (천천히 말하고 싶음)
└─ 독립심: 60점 (강요받기 싫음)

박간병인의 성격:
├─ 감정 표현: 30점 (업무만 집중)
├─ 활동 선호: 85점 (운동 많이 시킴)
├─ 인내심: 40점 (빠르게 일 처리)
└─ 독립심: 35점 (다 해드려야 함)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔴 차이점:
  감정: |80 - 30| = 50점 차이 → 궁합 50점
  활동: |30 - 85| = 55점 차이 → 궁합 45점
  인내: |85 - 40| = 45점 차이 → 궁합 55점
  독립: |60 - 35| = 25점 차이 → 궁합 75점

  평균 궁합: 56점 (C등급)
```

### 💔 왜 성격 차이가 문제인가?

**하루에 일어나는 실제 충돌:**

```
[아침 8시] 약 먹는 시간
할머니: "이 약이 뭐였더라... 어제..."
간병인: (끊으며) "그냥 드세요, 늦었어요"
→ 할머니 마음: 상처 +1

[오전 10시] 청소 시간
할머니: "젊었을 때 이야기 좀..."
간병인: "바빠요, 나중에요"
→ 할머니 마음: 외로움 +1

[오후 2시] 운동 시간
할머니: "오늘은 피곤한데..."
간병인: "안 됩니다, 일어나세요"
→ 할머니 마음: 강요받는 느낌 +1

[저녁 6시] 저녁 식사
할머니: "제가 스스로 할게요..."
간병인: "제가 해드릴게요" (다 해줌)
→ 할머니 마음: 존중받지 못함 +1

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

하루에 작은 상처 10번 × 3일 = 30번
→ "바꿔주세요" 전화
```

## 이야기 2: 왜 이것이 "개인 문제"가 아니라 "사회 문제"인가?

### 📊 숫자로 보는 현실

```
🇰🇷 대한민국 현황 (2024년)

간병이 필요한 노인: 약 180만 명
현재 간병인: 약 40만 명

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

현재 매칭 방식의 결과:

1️⃣ 환자 불만족률: 40%
   → 72만 명이 불만족
   → 매일 외로움, 스트레스

2️⃣ 평균 재매칭 주기: 3개월
   → 1년에 4번 바꿈
   → 180만 명 × 4회 = 720만 건/년

3️⃣ 재매칭 1건당 비용: 50만원
   (새 사람 찾기 + 교육 + 빈자리 기간)
   → 720만 건 × 50만원 = 3조 6천억원/년

4️⃣ 간병인 이직률: 60%
   → 스트레스로 1년 안에 60% 그만둠
   → 인력 부족 악순환
```

### 💸 한 가정의 실제 비용

```

# 📖 Part 2: 왜 기존 해결책이 실패하는가?

## 방법 1: "경험 많은 담당자의 직관"

### 사례: 10년 경력 담당자 박대리

```
박대리: "저 10년 했어요. 보면 알아요."

김할머니 케이스:
박대리: "할머니가 조용하시네...
        조용한 간병인이 좋겠다.
        최간병인님이 말이 없으시니까 딱이다!"

→ 배치

3일 후:
할머니: "너무 조용해서 답답해요...
        제가 말을 걸어도 대답을 안 하시고..."

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ 왜 실패했나?

박대리가 본 것: "할머니가 말이 적다"
실제 진실: "할머니가 낯을 가린다"

할머니 실제 성격: 감정 교류 80점 (따뜻함 원함)
최간병인 성격: 감정 교류 20점 (업무만)

→ 박대리의 "직관"은 첫인상일 뿐
→ 진짜 성격은 12개 질문으로 파악해야 함
```

### 왜 사람의 직관은 실패하는가?

```
1️⃣ 편견 (Bias)
   "할머니는 다 비슷할 거야"
   "간병인은 다 친절할 거야"
   → 실제로는 천차만별

2️⃣ 제한된 정보
   담당자가 할머니를 만나는 시간: 10분
   실제 간병인과 지내는 시간: 24시간 × 365일
   → 10분으로 어떻게 1년을 예측?

3️⃣ 확인 편향
   "이 사람 경력 10년이니까 괜찮을 거야"
   → 경력과 성격은 별개
   → 하지만 경력만 보고 '괜찮다'고 확신

4️⃣ 피로 누적
   담당자가 하루에 매칭하는 건수: 10건
   → 오후 4시쯤 되면 "그냥 아무나"
   → 집중력 저하

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

결과:
담당자 10년 경력 vs 컴퓨터 1초
→ 컴퓨터가 더 정확함 (편견 없음, 피로 없음)
```

## 방법 2: "설문지로 매칭"

### 사례: A 병원의 시도

```
A 병원: "우리도 성격 테스트 해봤어요!"

환자용 설문지:
1. 성격이 어떠신가요?
   ○ 활발함  ○ 조용함

2. 어떤 간병인을 원하시나요?
   ○ 친절한  ○ 전문적인

3. 활동을 좋아하시나요?
   ○ 예  ○ 아니오

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ 결과: 여전히 40% 불만족

왜?
```

### 왜 간단한 설문은 실패하는가?

```
문제 1: 모호한 질문
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"성격이 어떠신가요?"
→ 할머니: "조용함" 선택

실제:
- 낯가림이 있어서 처음엔 조용함 (감정 표현 80점)
- 친해지면 대화 많이 원함

→ "조용함" 체크가 감정 표현 80점을 못 잡아냄

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

문제 2: 단순한 선택지
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"활동을 좋아하시나요?" 예/아니오

실제:
- 활동 선호도는 0-100점 스펙트럼
- 30점 = "가끔 가벼운 산책"
- 90점 = "매일 1시간 운동"

→ 예/아니오로는 구분 불가

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

문제 3: 원하는 것 vs 맞는 것
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
"어떤 간병인 원하세요?"
→ 할머니: "친절한 사람이요"

→ 모든 환자가 "친절"을 원함
→ 하지만 "친절"의 정의가 다름

할머니 A: 친절 = 대화 많이 들어줌 (감정 80점)
할머니 B: 친절 = 조용히 지켜봄 (감정 20점)

→ 같은 단어, 다른 의미
→ 매칭 실패
```

- "너무 어려워요"

### 왜 AI 면접이 실패하는가?

```
문제 1: 고령자 디지털 격차
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
82세 평균 스마트폰 사용 능력: 낮음
챗봇 대화: 더 어려움

→ 70% 이탈률

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

문제 2: 자유 응답의 불명확성
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
할머니: "좋은 분"
→ AI가 해석 불가
→ "좋은" = 감정 표현? 활동성? 인내심?

vs.

12개 객관식 질문:
"손을 잡아주는 것을 좋아하시나요?"
→ 명확하게 감정 표현 측정

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

문제 3: 일관성 없음
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
같은 할머니, 다른 날:
월요일: "활발한 분이 좋아요"
화요일: "조용한 분이 좋아요"

→ 기분에 따라 대답 변함
→ 점수화 불가능

vs.

구조화된 12개 질문:
→ 검증된 심리 측정 도구
→ 일관성 높음
```

# 📖 Part 3: 왜 우리 방식이 작동하는가?

## 핵심 통찰: "성격은 숫자로 측정 가능하다"

### 왜 수치화인가?

```
❌ 기존: "할머니는 조용하신 편이에요"

# 🎤 발표 시 핵심 메시지

## 오프닝: 문제의 고통 보여주기

```
"할머니가 3일 만에 간병인을 바꿔달라고 하셨습니다.

왜냐고요?

'저 분이 나쁜 건 아닌데...
 내가 말을 하려고 하면 끊으시고,
 쉬고 싶은데 자꾸 운동 시키시고...
 나는 천천히 말하고 싶은데 답답해하시고...'

경력 10년, 자격증도 있는 분인데 말이죠.

문제는 '성격'이었습니다.

그리고 이런 일이 1년에 720만 건 일어납니다.
비용으로 3조 6천억원입니다.

왜 이런 일이 반복될까요?"
```

## 본론: 왜 우리 방식이 다른가

```
"기존 방식은 '경력'만 봤습니다.

하지만 경력과 성격은 별개입니다.

10년 경력이어도 성격이 안 맞으면
3일 만에 교체됩니다.

우리는 '성격'을 봅니다.

4가지 축으로 수치화하고,
컴퓨터가 1,000명 중에서
가장 잘 맞는 사람을 찾습니다.

왜 컴퓨터인가?

사람은 편견이 있습니다.
사람은 피로가 있습니다.
사람은 5명만 비교합니다.

컴퓨터는 편견 없이,
피로 없이,
1,000명 전체를 1초에 비교합니다.

결과?
62% → 87% 정확도.
25% 향상입니다."
```
