```

## 💡 바른케어의 해결 방법: "성격도 봅니다"

### 우리는 이렇게 합니다:

```
🔍 1단계: 할머니의 성격을 이해합니다
   → 질문 12개로 할머니가 어떤 분인지 파악

🔍 2단계: 1,000명 간병인의 성격 정보를 가지고 있습니다
   → 각 간병인이 어떤 스타일인지 미리 알고 있음

🤝 3단계: "이 할머니랑 성격 잘 맞는 분" 찾아드립니다
   → 컴퓨터가 자동으로 가장 잘 맞는 10명 추천

결과:
✅ 할머니와 말 속도가 비슷한 간병인
✅ 할머니가 원하는 만큼 감정 교류 해주는 간병인
✅ 1년째 만족스럽게 지내는 중
```

**핵심:** 경력 + 의료 능력 뿐 아니라 **"성격 궁합"**까지 봅니다!

## 🧩 성격을 어떻게 파악하나요?

### 4가지 질문으로 성격 파악

마치 MBTI처럼, 우리는 **4가지 성격 특징**을 봅니다:

```
1️⃣ 감정 표현 스타일
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   무뚝뚝 ←────────────────→ 따뜻함

   예시:
   • 낮음: "괜찮아요, 혼자 있을게요"
   • 높음: "손 좀 잡아주실래요? 외로워요"

2️⃣ 활동 선호도
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   조용히 쉬기 ←─────────────→ 움직이기

   예시:
   • 낮음: "TV 보면서 침대에 누워있고 싶어요"
   • 높음: "산책 가고 싶어요, 복도라도 걷고 싶어요"

3️⃣ 반복 설명 필요
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   빠르게 ←───────────────→ 천천히

   예시:
   • 낮음: "한 번만 들어도 이해해요"
   • 높음: "여러 번 설명해주세요, 천천히요"

4️⃣ 독립심
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   도움 많이 필요 ←─────────→ 혼자 하고 싶음

   예시:
   • 낮음: "다 도와주세요, 혼자는 무서워요"
   • 높음: "가능하면 제가 할게요, 지켜만 봐주세요"
```

**이 4가지로 환자와 간병인 모두를 파악합니다.**

## 🔢 점수는 어떻게 계산하나요? (초간단 버전)

### 단계별 설명

#### 1️⃣ 각 성격 항목의 "차이"를 봅니다

```
김할머니 감정 표현: 80점
이간호사 감정 표현: 75점
차이: 5점 → "거의 같네!" → 95점 궁합

김할머니 감정 표현: 80점
박간병인 감정 표현: 30점
차이: 50점 → "너무 달라!" → 50점 궁합
```

**쉽게 말하면:**

- 차이 작음 = 궁합 좋음 = 높은 점수
- 차이 큼 = 궁합 나쁨 = 낮은 점수

#### 2️⃣ 4개 항목의 평균을 냅니다

```
이간호사 vs 김할머니:
  감정 궁합: 95점
  활동 궁합: 95점
  반복 궁합: 95점
  독립 궁합: 95점
  ──────────────
  평균: 95점 ← "성격 궁합 점수"
```

#### 3️⃣ 의료 능력(40%)과 성격 궁합(60%)을 합칩니다

```
최종 점수 = (의료 능력 × 40%) + (성격 궁합 × 60%)

예시 1: 이간호사
  의료: 100점 × 40% = 40점
  성격: 95점 × 60% = 57점
  ─────────────────────
  최종: 97점 → A+ 등급

예시 2: 박간병인
  의료: 66점 × 40% = 26.4점
  성격: 54점 × 60% = 32.4점
  ─────────────────────
  최종: 58.8점 → C 등급
```

**왜 40:60 비율인가?**

```
의료 능력 40%: "기본 조건"
  → 간호사라면 고혈압 관리할 수 있는가?
  → 이건 최소한 충족되어야 함

성격 궁합 60%: "만족도 결정"
  → 의료 기술은 가르칠 수 있지만
  → 성격은 바꿀 수 없음
  → 1년 동안 매일 얼굴 보는데 성격이 더 중요
```

## 📊 등급은 뭔가요?

```
100점────────────────────────────────────
       │
  95점 ├─ A+  ← 이상적! 거의 완벽한 궁합
       │        (1000명 중 상위 5%)
       │
  85점 ├─ A   ← 우수! 추천합니다
       │        (1000명 중 상위 20%)
       │
  75점 ├─ B+  ← 양호, 괜찮아요
       │        (1000명 중 상위 40%)
       │
  65점 ├─ B   ← 보통, 기본은 해요
       │        (1000명 중 상위 60%)
       │
   0점 ├─ C   ← 재고려 필요
       │        (다른 분 알아보시는 게...)
       │
───────────────────────────────────────────
```

**투자자가 기억할 것:**

- A+/A 등급 = 환자 만족도 85% 이상
- C 등급 = 3일 안에 교체 요청 확률 높음

## 💎 왜 이게 혁신적인가요?

### 기존 병원/기관의 방식

```
❌ 경력 5년 이상? → OK, 배치
❌ 간호사 자격증? → OK, 배치
❌ "누가 비었지?" → 그 사람 배치

결과:
→ 환자 불만족 30~40%
→ 간병인 이직률 높음 (스트레스)
→ 3개월마다 재매칭 필요
```

### 바른케어의 방식

```
✅ 경력 ✓ + 자격증 ✓ + 성격 궁합 ✓
✅ 1,000명 중에서 가장 잘 맞는 사람 찾기
✅ A+ 등급만 추천

결과:
→ 환자 만족도 85% 이상
→ 간병인도 만족 (스트레스 ↓)
→ 1년 이상 장기 매칭 가능
```

## 🚀 경쟁력: 숫자로 증명

### 시간 절감

```
기존 방식:
  👤 담당자가 1000명 리스트 보면서
  👤 "음... 이 사람 괜찮을까?"
  👤 10~20명 추리고 → 전화하고 → 면접 잡고
  ⏱️ 소요 시간: 1~2주

바른케어:
  💻 컴퓨터가 1초 안에 1000명 분석
  💻 가장 잘 맞는 10명 자동 추천
  ⏱️ 소요 시간: 1초

📈 효율성: 99.9% 시간 단축
```

### 비용 절감

```
재매칭 비용:
  기존: 3개월마다 1번씩 = 연 4회
  바른케어: 1년에 1번 또는 안 함

  환자 1명 재매칭 비용: 약 50만원
  연간 절감액: 150만원 × 환자 수

  예: 100명 환자 기관
  → 연간 1억 5천만원 절감
```

### 만족도 향상

```
기존 방식:
  만족도: 60%
  재매칭 요청: 40%
  1년 유지율: 30%

바른케어:
  만족도: 85%
  재매칭 요청: 15%
  1년 유지율: 70%

📈 만족도 25% 향상

## 🎯 투자자 FAQ

### Q1: 왜 사람이 아니라 컴퓨터가 더 잘 매칭하나요?

```
A: 사람은 편견이 있습니다.

담당자: "이 사람 경력 5년이고 착해 보이네, 배치!"
       ↑ 성격 궁합은 안 봄

컴퓨터: "1,000명의 4가지 성격 항목을 모두 비교"
       "환자와 차이가 가장 적은 사람 찾기"
       ↑ 편견 없이 객관적

결과: 컴퓨터가 더 정확함
```

### Q2: 성격이 정말 중요한가요? 의료 능력이 더 중요한 거 아닌가요?

```
A: 의료 능력은 "최소 조건"이고, 성격은 "만족도 조건"입니다.

비유:
  의료 능력 = 운전면허
  성격 궁합 = 운전 스타일

  당신이 택시를 탑니다:
  - 운전면허는 당연히 있어야 함 (최소 조건)
  - 하지만 난폭운전 vs 안전운전 = 만족도 결정

간병도 마찬가지:
  - 의료 자격은 당연히 있어야 함 (40%)
  - 하지만 1년 동안 매일 보는 사람이라면
    성격이 더 중요함 (60%)

실제 데이터:
  의료 능력 100점 + 성격 40점 = 64점 (C등급)
  → 3일 안에 "바꿔주세요"

  의료 능력 80점 + 성격 90점 = 86점 (A등급)
  → 1년 넘게 만족
```

### Q3: 왜 4가지 항목인가요?

```
A: 간병 만족도를 결정하는 가장 중요한 4가지입니다.

1. 감정 표현: "따뜻하게 대해주나요?"
   → 환자 만족도 1순위

2. 활동성: "운동 많이 시키나요?"
   → 환자 스트레스 원인 1순위

3. 반복 설명: "천천히 설명해주나요?"
   → 노인 환자 필수 요소

4. 독립심: "다 해주나요, 지켜만 보나요?"
   → 환자 존중감 결정

4가지면 충분히 성격 파악 가능
(너무 많으면 복잡하고, 적으면 정확도 ↓)
```

### Q4: 1,000명 간병인 데이터를 어떻게 구했나요?

```
A: 실제 간병인들에게 동일한 12개 질문으로 성향 테스트를 진행했습니다.

과정:
1. 간병인 등록 시 성향 테스트 진행 (5분 소요)
2. 자동으로 4가지 점수 계산
3. 데이터베이스에 저장
4. 환자 매칭 시 자동으로 비교

업데이트:
- 6개월마다 재테스트 (성향 변화 반영)
- 환자 피드백으로 보정
```

### Q5: 다른 회사도 이렇게 할 수 있지 않나요?

```
A: 3가지 진입 장벽이 있습니다.

1️⃣ 알고리즘 설계
   → 4가지 축을 정하고
   → 40:60 비율을 찾고
   → 정확도를 검증하는데 6개월 걸림

2️⃣ 데이터 확보
   → 1,000명의 간병인 성향 데이터 필요
   → 각 간병인당 12개 질문 = 12,000개 데이터 포인트
   → 우리는 이미 확보 완료

3️⃣ 검증
   → 실제 매칭 결과로 정확도 입증 필요
   → A+ 등급 만족도 85% 이상 확인
   → 이 데이터가 핵심 경쟁력
```
